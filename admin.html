Connect Google Sheets

The tracking system needs a SheetDB API URL to store and read shipment data. Follow the steps below to set it up. This takes about 5 minutes.

Step 1 — Create the Google Sheet

Go to sheets.google.com in the client's Google account. Create a new spreadsheet named QuickFetch Shipments. In Row 1, add these exact column headers:

tracking_number | full_name | phone | email | freight_type | origin | destination | route_code | weight | status | update_title | update_location | update_activity | update_time | tracking_history | created_date | notes

Step 2 — Connect to SheetDB

Go to sheetdb.io, sign up for free, click "Create API", and connect the Google Sheet you just created. SheetDB will give you an API URL that looks like:

https://sheetdb.io/api/v1/xxxxxxxxxxxxxxx

Step 3 — Paste the URL below

Copy that API URL from SheetDB and paste it here to activate the system.
